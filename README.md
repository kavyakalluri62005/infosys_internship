# AI-Powered Market Trend & Consumer Sentiment Forecaster


# Milestone 1 - Data Pipeline Setup

## Project Overview
This project focuses on building a basic data scraping and ingestion pipeline.

## What I Did
- Scraped laptop product data from a public test e-commerce website
- Extracted product name, price, ratings
- Generated sentiment labels (Positive, Negative, Neutral)
- Stored cleaned data in CSV format

## Tools Used
- Python
- Requests
- BeautifulSoup
- CSV

## Output
- Structured CSV dataset created
- Contains product name, price, ratings, sentiment, and date
- Ready for analysis in next milestone
